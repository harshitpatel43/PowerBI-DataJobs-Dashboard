# PowerBI-DataJobs-Dashboard
Interactive Power BI dashboard project focused on analyzing global data job trends, salary insights, hiring patterns, and role-based comparisons using dynamic visualizations, KPI cards, slicers, and drill-through analysis.

## How to use
Right-click on a specific Job Role in the main chart and select 'Drill-through' to see a detailed breakdown of salary by experience level and location.

## Project Overview

This project is an interactive Power BI dashboard built to analyze trends in the Data Jobs market.
The dashboard provides insights into:

-Total job opportunities
-Average yearly salary
-Average hourly salary
-Salary comparison between different data roles
-Job trends over time
-Highest paying data-related jobs

The purpose of this project is to demonstrate:

-Data visualization
-Dashboard design
-Interactive reporting
-Drill-through analysis
-Power BI development skills
